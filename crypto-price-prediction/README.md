# Cryptocurrency Price Prediction using LSTM

## Overview
This project builds a time-series forecasting model that predicts future cryptocurrency prices using a Long Short-Term Memory (LSTM) neural network.

The model learns patterns from historical market data (Bitcoin in this project) and predicts the next day's closing price based on the previous 60 days of prices.

This project demonstrates the application of Deep Learning in financial data analysis and forecasting.

---

## Features
- Collects historical cryptocurrency price data
- Cleans and normalizes the dataset
- Converts data into time-series sequences
- Trains an LSTM neural network
- Predicts future prices
- Evaluates prediction accuracy (MAE & RMSE)
- Visualizes actual vs predicted price trends
- Forecasts next-day price

## Dataset
Historical cryptocurrency price data is fetched from Yahoo Finance using the `yfinance` library.

Dataset attributes:
- Date
- Open
- High
- Low
- Close
- Adjusted Close
- Volume

Only the **Close price** is used for prediction because it represents the final market value for the day.

---

## Methodology

### 1. Data Collection
The project downloads Bitcoin historical price data using the Yahoo Finance API.

### 2. Data Preprocessing
- Missing values are removed
- Only closing price column is selected
- Data is normalized between 0 and 1 using MinMaxScaler

Normalization helps neural networks train faster and more accurately.

### 3. Sequence Creation
The model uses a sliding window technique:

**Previous 60 days → Predict next day price**

This helps the LSTM learn temporal dependencies in price movement.

### 4. LSTM Model Architecture
The neural network consists of:
- LSTM Layer (pattern learning)
- Dropout Layer (prevent overfitting)
- LSTM Layer
- Dropout Layer
- Dense Output Layer

Compilation:
- Optimizer: Adam
- Loss Function: Mean Squared Error (MSE)

### 5. Training
The dataset is divided into:
- 80% Training Data
- 20% Testing Data

The model learns from training data and is validated using unseen test data.

### 6. Prediction
After training, the model:
- Predicts prices for the test dataset
- Forecasts the next day's closing price

### 7. Evaluation Metrics
Two metrics are used:

**MAE (Mean Absolute Error)**  
Average difference between predicted and actual price.

**RMSE (Root Mean Squared Error)**  
Measures model accuracy and penalizes large errors.

Lower values indicate better performance.

### 8. Visualization
The project generates:
- Historical price graph
- Actual vs Predicted price comparison
- Zoomed recent comparison
- Next day prediction graph
